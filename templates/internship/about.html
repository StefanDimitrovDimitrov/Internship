Overview

I am a web developer with experience with Python, and JS. I recently completed the training path in Software University and this (Django) web application is the result of my final project in the academy.

INternSHIP is a job ad web application concentrated in helping junior specialists find their first job in the field they want to develop in.

The main functionalities of the application are:
  • Creating Company and Candidate accounts
  • Publishing job ads
  • Reviewing candidates
  • Uploading CVs, applying for positions
  • Authentication and Authorization for the different roles of the application
  • Admin panel
  • Reviewing catalog of companies and job ads

Tools used for building the web application:
  • Python
  • Django
  • PostgreSQL
  • Bootstrap 5
  • HTML
  • CSS
  • Heroku for deployment
